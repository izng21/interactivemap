## 📖 Overview

The **PUP Maragondon Campus Interactive Map** is a lightweight, browser-based campus navigation tool built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. It provides students, faculty, and visitors with an intuitive way to explore the campus, view building details, and browse floor plans directly from any device.

> 🎓 Developed for the Polytechnic University of the Philippines – Maragondon Campus.

---

## ✨ Features

- 🏛️ **Interactive Campus Map** — Click on buildings to view their details
- 🗂️ **Floor Plan Viewer** — Browse floor plans for each campus building
- 🏢 **Building Detail Pages** — Dedicated info pages per building/facility
- 📱 **Responsive Design** — Works seamlessly on desktop and mobile
- ⚡ **No Dependencies** — Pure HTML/CSS/JS, no frameworks required
- 🚀 **Fast & Lightweight** — Instant load, no build step needed

## 🚀 Getting Started

No installation needed. Just clone and open!

```bash
# 1. Clone the repository
git clone https://github.com/izng21/interactivemap.git

# 2. Navigate into the project folder
cd interactivemap

# 3. Open index.html in your browser
open index.html
```

Or simply **[visit the live demo](https://pupmcinteractivemap.vercel.app)** — no setup required.
